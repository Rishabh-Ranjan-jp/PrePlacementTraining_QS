# Placement Training Notes & Code

This repository contains my session notes, in-class coding exercises, and assigned homework from pre-placement training. The training covers multiple programming languages and technical concepts, and the repository is structured to scale cleanly as new languages are introduced.

---

## 📂 Repository Structure

The repository is organized first by **programming language**, and then by **content type** and **session date**.

The directory tree follows this general pattern:
`/<language>/<category>/<session_folder>/`

**Categories:**

* **`/<language>/notes/`** Markdown files containing detailed lecture notes, syntax rules, and core concepts.

* **`/<language>/exercises/`** Source code for the specific examples and exercises executed alongside the trainer during class.

* **`/<language>/assignments/`** Source code for the homework and logic-building problems assigned at the end of each session.

**Current languages tracked:**

* `/java/`

---

## 🚀 How to Run the Code

**For Java (`/java/...`):**

1. Navigate to the specific directory containing the `.java` file.

2. Compile the code using `javac`:
   
   ```bash
   javac filename.java
   ```

3. Run the compiled program: 
   
   ```bash
   java filename
   ```
